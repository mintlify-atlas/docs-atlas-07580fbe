Future is a TypeScript library that brings structured concurrency and full lifecycle control to asynchronous code. Unlike native Promises, a `Future` can be paused, resumed, cancelled, composed with other Futures, and used in `for await...of` loops — all while remaining fully compatible with `await`.

## Why Future?

Native JavaScript Promises have well-known limitations: once started, they cannot be cancelled or paused. `Future` solves this with a generator-based execution model that gives you fine-grained control.

<CardGroup cols={2}>
  <Card title="Cancellable" icon="ban">
    Cancel any in-flight Future at any time. Cancellation propagates automatically via `AbortController`.
  </Card>
  <Card title="Pausable" icon="pause">
    Pause and resume execution at yield points without losing state.
  </Card>
  <Card title="Composable" icon="layer-group">
    Combine Futures with `all`, `race`, `allSettled`, `scope`, and `withConcurrencyLimit`.
  </Card>
  <Card title="Promise-compatible" icon="check">
    Use `await`, `.then()`, `.catch()`, and `.finally()` — Futures work wherever Promises do.
  </Card>
</CardGroup>

## Get started in 3 steps

<Steps>
  <Step title="Install the package">
    Add `@okikio/future` to your project via JSR or npm.

    <CodeGroup>
    ```bash npm
    npm install @okikio/future
    ```
    ```bash Deno (JSR)
    deno add jsr:@okikio/future
    ```
    ```bash Bun
    bun add @okikio/future
    ```
    </CodeGroup>
  </Step>

  <Step title="Create your first Future">
    Use `from()` to wrap an async generator function into a Future.

    ```typescript
    import { from } from "@okikio/future";

    const future = from(async function* () {
      yield 1;
      yield 2;
      return 3;
    });

    for await (const value of future) {
      console.log(value); // Logs 1, 2
    }
    ```
  </Step>

  <Step title="Control execution">
    Pause, resume, or cancel your Future at any point.

    ```typescript
    import { from } from "@okikio/future";

    const future = from(async function* () {
      yield 1;
      yield 2;
      return 3;
    });

    future.pause();
    setTimeout(() => future.resume(), 500);

    // Or cancel it entirely
    await future.cancel();
    ```
  </Step>
</Steps>
