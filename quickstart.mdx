---
title: "Get started with Future"
description: "Create your first Future, iterate yielded values with for-await, pause and resume execution, and cancel async operations — all in under 5 minutes."
---

Future gives you cancellable, pausable async operations built on async generators. This guide walks you from installation to a working example with pause, resume, and cancellation in a few steps.

<Steps>
  <Step title="Install the package">
    <CodeGroup>
    ```bash npm
    npm install @okikio/future
    ```
    ```bash Bun
    bun add @okikio/future
    ```
    ```bash pnpm
    pnpm add @okikio/future
    ```
    ```bash Deno (JSR)
    deno add jsr:@okikio/future
    ```
    </CodeGroup>
  </Step>

  <Step title="Create your first Future">
    Use `from()` to wrap an async generator function into a Future.

    ```typescript
    import { from } from "@okikio/future";

    const future = from(async function* () {
      yield 1;
      yield 2;
      return 3;
    });
    ```
  </Step>

  <Step title="Iterate yielded values">
    Use `for await...of` to consume values as the Future runs.

    ```typescript
    for await (const value of future) {
      console.log(value); // Logs 1, then 2
    }
    ```
  </Step>

  <Step title="Await the final result">
    Use `toPromise()` to get the return value, or `await` directly — Future is Promise-compatible.

    ```typescript
    const result = await future.toPromise(); // 3

    // Or use await directly
    const result2 = await future; // 3
    ```
  </Step>

  <Step title="Pause and resume">
    Suspend execution at the next yield point and resume it later.

    ```typescript
    import { from } from "@okikio/future";

    const future = from(async function* () {
      yield 1;
      yield 2;
      return 3;
    });

    future.pause();
    setTimeout(() => future.resume(), 500);

    for await (const value of future) {
      console.log(value); // Resumes after 500ms
    }
    ```
  </Step>

  <Step title="Cancel a Future">
    Cancel stops execution and cleans up resources. By default it throws a `CancellationError`.

    ```typescript
    import { from, CancellationError } from "@okikio/future";

    const future = from(async function* () {
      yield 1;
      yield 2;
      return 3;
    });

    try {
      // Start iteration then cancel
      const iterator = future[Symbol.asyncIterator]();
      await iterator.next(); // { value: 1, done: false }
      await future.cancel();
    } catch (e) {
      if (e instanceof CancellationError) {
        console.log("Future was cancelled");
      }
    }
    ```
  </Step>
</Steps>

## Real-world example

Combine `withDeadline` and `withAbortable` to fetch data with a timeout and external cancel support:

```typescript
import { from, withDeadline, withAbortable } from "@okikio/future";

const controller = new AbortController();

const future = withDeadline(
  withAbortable(
    from(async function* () {
      const response = await fetch("https://api.example.com/data", {
        signal: controller.signal,
      });
      return await response.json();
    }),
    controller,
  ),
  5000, // Cancel automatically after 5 seconds
);

// Cancel from user action
document.querySelector("#cancel")?.addEventListener("click", () => {
  controller.abort();
});

try {
  const data = await future.toPromise();
  console.log(data);
} catch (e) {
  console.error("Request failed or timed out:", e);
}
```

<Tip>
  You can also use `await using future = from(...)` to automatically cancel and clean up the Future when the enclosing block exits.
</Tip>
